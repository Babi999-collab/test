# Contribution Test Project

This repository is a simple project created to practice collaboration using Git and GitHub.

## Purpose

This is a contribution test focused on implementing and improving simple mathematical functions.

## Features

- Basic math operations (addition, subtraction, multiplication, division)
- Easy to understand code for beginners
- Practice environment for collaboration and pull requests

## Goal

The main goal of this project is to learn:
- Git workflows
- Collaboration using branches and pull requests
- Code review and version control

## Contributions

Contributions are welcome! Feel free to:
- Add new mathematical functions
- Improve existing code
- Fix bugs
- Suggest enhancements
